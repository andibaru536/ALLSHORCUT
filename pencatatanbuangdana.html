Cara Penggunaan:

  • Tempel atau ketik data mentah di area input (bisa beberapa data sekaligus)
  • Klik tombol "Proses Data"
  • Output akan ditampilkan di bagian bawah
  • Klik tombol "Salin Hasil" untuk menyalin data
  • Gunakan tombol "Hapus/Reset" untuk mengosongkan semua data
Contoh format input (beberapa data sekaligus): Data 1: Info : Goltogel Perihal : Pengembalian Dana Dari Situs : Linetogel Bank Danamon ( BERSIH ) Nama Rekening : ZINTIYA AMELYA MBATU Nomor Rekening : 3692383056 Nominal : Rp. 50,000,000,- Data 2: Bank BCA Nama: JOHN DOE No. Rek: 1234567890 Jumlah: Rp 1.500.000 Data 3: BANK MANDIRI NAMA: JANE SMITH NOMOR REKENING: 9876543210 NOMINAL: RP 2.750.000
